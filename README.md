# ForecastFM

ForecastFM is a Java-based mashup service that generates Spotify playlists based on current weather conditions using external APIs.

## Overview

The application retrieves real-time weather data and maps it to music mood parameters in order to create context-aware Spotify playlists.

## Tech Stack

- Java
- OpenWeather API
- Spotify Web API
